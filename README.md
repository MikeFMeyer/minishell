## Installation:  
  
Clone the repo:  
  
```
git clone https://github.com/mikefmeyer/minishell  
cd Minishell/Minishell
```  
Run the following to build the binary: ```make```  
  
## Running:  

#### Run the shell with:  
```./minishell```  

#### Exit the shell with:  
```exit```  
  
#### Cancel current process with:  
```Ctrl + C```
